INSTALLATION & SETUP

Manual installation is easy and takes fewer than one minute.

  1. Download the plugin from CodeCanyon, unpack it and upload the [Contact Form 7 Database] folder to your wp-content/plugins/ directory.
  2. Activate the plugin through the ‘Plugins‘ menu in WordPress.
  3. Go to your main WordPress left menu > Contact > Database to configure the basic options and click Save Changes button.
  4. You’re done. Enjoy.
